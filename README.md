# eDPI Calculator & Sensitivity Converter

FPS 게임 유저를 위한 간편 eDPI 계산기 및 게임별 감도 변환 도구입니다.
자신의 마우스 설정값을 최적화하여 더 정확한 에이밍을 경험하세요.

## 🚀 주요 기능
* eDPI 계산 : 마우스 DPI와 인게임 감도를 입력하여 정확한 eDPI를 계산합니다.
* 시각화 게이지 : 결과값에 따라 감도 수준을 게이지 바(Gauge Bar)로 직관적으로 표시합니다.
* 발로란트 감도 변환 : 타 게임 감도를 발로란드 기준 감도로 자동 변환해 줍니다.
* 반응형 디자인 : PC와 모바일 환경 어디서든 편리하게 사용할 수 있습니다.
